##Le frontend (qui contient donc du dummy content) a été conçu avec React
##Vous pouvez télécharger l'archive frontend et l'installer en local pour la visiter:

0-Avoir Node (NPM) installé
1-Télécharger l'archive et la décompresser
2-Ouvrir votre terminal ou IDE à la racine du projet et lancer un "npm install" pour installer les modules complémentaires Node.
3-Tapez la commande "npm start" une fois l'installation des packs Node terminée, vous pourrez parcourir le frontend de l'application en local. (localhost:3000)
